### Sobre
Repositório do projeto de testes automatizados do sistema starbugs, feito no curso Cucumber eXperience!

### Tecnologias
* Capybara
* Cucumber
* RSpec
* Ruby
* Selenium Webdriver


### Como executar
1. Clonar o repositório
```
bundle install
```
2. Executar testes
```
bundle exec cucumber
```
